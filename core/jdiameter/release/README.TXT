--------------
 Introduction
--------------

 This is the stack release of Mobicents Diameter solution.

 This stack is based on the java.net project leaded by Erick Svensson, named jDiameter.

 It enables the user to develop highly effective standalone and enterprise (J2EE & JSLEE) java applications working in 
 diameter networks using SSL encryption. The basis for all component serves is the jDiameter API. jDiameter API also
 can be used for commercial realizations of a stack and appendices such as NASREQ (RFC 4005), EAP-based authentication
 (RFC 4072), Credit-Control (RFC 4006), or even some of the 3GPP applications (Sh, Ro, Rf, etc.).

------------------
 Release Contents
------------------

  jdiameter-api-@JDIAM_VERSION@.jar (contains the compiled classes for the API)

  jdiameter-impl-@JDIAM_VERSION@.jar (contains the compiled classes for the Implementation)


----------------
 How to Use it?
----------------

 In order to use it, just import the compiled classes jars for the API and Implementation as a library to your project.
